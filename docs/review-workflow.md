# Human Review Workflow

## Statuses

- `new`
- `in_review`
- `needs_more_evidence`
- `approved`
- `approved_with_edits`
- `rejected`
- `deferred`
- `exported_to_odk`
- `odk_validation_failed`
- `merged`

## Export Rule

Only these statuses may be exported:

- `approved`
- `approved_with_edits`

Confidence scores never imply approval.
